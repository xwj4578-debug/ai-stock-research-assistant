# Development Guide

AlphaLens AI 的开发顺序固定为：

```text
PDS -> UI -> Database -> API -> Prompt -> Backend -> Frontend -> Test -> Review -> Release
```

原则：

- Research First
- Explainable AI
- AI Assist, Human Decide
- Documentation Driven Development
- docs 为需求唯一来源
- API First
- Testability
- Extensibility
