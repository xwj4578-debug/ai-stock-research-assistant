# Architecture Principles

## Principles

- Modular Architecture
- API First
- Testability
- Extensibility
- Open Source Friendly

## Boundaries

- `docs/`：需求和设计真源
- `api/`：接口契约
- `backend/`：后端实现
- `frontend/`：正式前端
- `prototype/`：早期原型
- `agents/`：Agent 规划和实现
- `prompts/`：Prompt 模板
- `database/`：数据模型和迁移
