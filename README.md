# AI 股票研究助手

这是一个按 `AI 股票分析平台开发计划.pdf` 实现的 V1 MVP。产品定位不是行情软件，而是研究助手：输入公司后，快速输出业务理解、商业模式、财务趋势、估值判断、重大消息、同行对比、风险点和透明评分依据。

## 当前能力

- 首页综合排行榜：评分、行业、PE、PB、ROE、涨跌幅、风险等级。
- 公司研究页：公司简介、所属板块、商业模式、五年财务趋势、估值分析、最近30天消息、风险、分红、股东、行业和同行对比。
- 多股票搜索：当前内置工业富联、浪潮信息、中际旭创、新易盛、贵州茅台、宁德时代、比亚迪、中芯国际、招商银行、中国平安、东方财富、科大讯飞、立讯精密、立昂微等样例。
- 全A股股票池：本地深度样例库未收录时，会自动查询东方财富A股股票池，支持按代码或名称打开任意A股研究框架页。首次查询会拉取全量股票池并缓存，后续查询会明显变快。
- 实时行情刷新：公司详情页会尝试从东方财富公开行情接口刷新实时价、当日涨跌幅、PE、PB、市值和刷新时间；失败时自动回退到本地样例数据。
- AI自动研究：公司页点击“AI自动研究”，后端会自动收集股票池、实时行情、公告标题等资料，生成结构化摘要、估值快照、近期公告、风险点、研究评分和来源。
- 五年财务趋势：动态A股会自动拉取东方财富F10主要财务指标，填充近五年营收、归母净利润、毛利率、净利率、ROE、经营现金流和资产负债率。
- 主营业务识别：动态A股会自动拉取东方财富F10公司资料，填充公司简介、主营业务、主要产品、客户类型、行业和概念标签。
- V1优化项：公司Header增加AI结论、合理估值区间、风险等级、更新时间和公司定位；评分按区间变色；新增投资逻辑状态、利好因素、利空因素、长期跟踪判断；消息按公告/新闻/评级/政策分类；同行表支持点击切换；支持浏览器打印导出PDF。
- 未收录股票兜底：输入本地库没有的股票时，不再报错，会生成“待AI搜索”的研究框架页。
- 透明评分：100分拆成商业模式、行业景气、盈利能力、成长能力、估值水平、财务健康、风险控制、消息面，每项都能展开查看依据。
- AI问答入口：基于本地研究数据库回答常见问题。

## 运行

```powershell
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

打开：

```text
http://127.0.0.1:8000
```

## 测试

```powershell
.\.venv\Scripts\python.exe -m pytest
```

## 数据口径

当前内置商业模式、风险、新闻等深度研究数据用于产品原型和评分逻辑验证，不构成投资建议。全A股代码、名称、行情、估值、公司资料、公告标题和近五年核心财务指标已加入自动刷新/自动研究层；新闻正文、研报正文、行业深度比较和宏观数据仍建议继续接正式数据源或LLM抽取。

下一步建议接入 `AI搜索任务`：用户输入股票后，系统自动检索公司官网、年报、公告、新闻、研报摘要和同行资料，再把结果写入结构化研究库。
