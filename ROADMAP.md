# Roadmap

## v0.1 Foundation

- 规范仓库目录。
- 明确 MVP 边界。
- 建立产品、设计、数据库、API、Prompt、Agent 文档入口。
- 保留可运行 FastAPI 原型。

## v0.2 Sprint 1 Design System

- 建立 Design Token。
- 建立 Typography、Spacing、Radius、Shadow 规范。
- 建立基础组件文档。
- 建立前端工程目录入口。

## v0.3 Product Design

- 输出产品设计说明书第一卷。
- 完成市场首页和股票详情页信息架构。
- 定义观察池、评分、AI 报告和聊天流程。

## v0.4 UI Foundation

- 建立 Design System。
- 设计首页、板块排行、股票筛选器、股票详情、观察池页面。
- 准备 Next.js + Tailwind 前端工程。

## v0.5 Backend Foundation

- 拆分 API 模块。
- 引入 PostgreSQL 数据模型。
- 引入 Redis 缓存。
- 标准化行情、公告、财务数据接口。

## v0.6 AI Agent MVP

- 实现 Research、Technical、Fundamental、News、Sector、Risk、Scoring、Report Agent 的基础编排。
- 接入 RAG 和报告生成模板。
